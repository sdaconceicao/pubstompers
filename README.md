# README #

Pubstompers hub site. Built with Angular.

### Details ###

* Uses Angular
* Version 1.0

### Setup ###

1. Run npm install
2. Edit app.conf.json with your team id and year. Mark current year with current = true;

### Startup ###
1. Run npm start

### Build ###
1. Run npm run build
2. Results output to dist directory
